## STRATÉGIE FRANCE BLEU PICARDIE

France Bleu est LA radio locale de référence. Plusieurs approches :

### 1. Spot publicitaire classique
- **Coût :** ~100-150€ par passage
- **Pack recommandé :** 10-15 passages sur 2 semaines
- **Budget :** 400-600€
- **Période :** Juin, 2 semaines avant le festival

### 2. Interview / chronique culture (GRATUIT)
- Contacter la rédaction locale
- Proposer une interview sur le festival
- Angle : "Festival local accessible, 4ème édition"
- Période : Mai ou début juin

### 3. Partenariat média
- France Bleu devient partenaire officiel
- En échange : visibilité sur vos supports
- Ils diffusent des spots gratuits + couverture éditoriale
- **C'est la meilleure option si vous l'obtenez**

## PRODUCTION DU SPOT

### Option A : Faire soi-même
- Enregistrer avec un bon micro
- Utiliser Audacity (gratuit) pour le montage
- Ajouter une musique libre de droits metal
- Coût : 0€

### Option B : Studio local
- Voix pro + mixage
- Coût : 100-200€

### Option C : Via la radio
- Certaines radios produisent le spot
- Inclus dans le pack pub
- Demander un devis

**Recommandation :** Option A pour les radios web, Option B ou C pour France Bleu
